### Function description

1 輸入框輸入後按下**新增**或是**Enter**皆可以新增至清單

2 在事情上面**點一下**或是按下**完成**皆會出現橫線

3 按下**更改**則可以呼叫出輸入框，即可以開始更改該項目

4 更改輸入完成後按下**更改**或是**Enter**後，項目就會立刻變動
